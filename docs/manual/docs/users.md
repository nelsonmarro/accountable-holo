# Gestión de Usuarios

Verith permite tener múltiples usuarios con diferentes niveles de acceso, ideal para negocios con cajeros o contadores.

## Roles Disponibles

*   **Administrador:** Tiene control total. Puede ver reportes financieros, cambiar la configuración del SRI y gestionar otros usuarios.
*   **Estándar:** Puede registrar transacciones y gestionar clientes, pero no tiene acceso a la configuración sensible ni a la eliminación de datos críticos.

## Crear un Usuario

1.  Ve a la pestaña **"Usuarios"** (Solo visible para Administradores).
2.  Haz clic en **"Nuevo Usuario"**.
3.  Ingresa:
    *   **Nombre de Usuario:** Para el inicio de sesión.
    *   **Contraseña:** Clave segura.
    *   **Rol:** Selecciona "Admin" o "Standard".
4.  Guarda los cambios.

## Seguridad

Si olvidas la contraseña de administrador, deberás contactar al soporte técnico para un restablecimiento seguro, ya que las claves están encriptadas en la base de datos local.
