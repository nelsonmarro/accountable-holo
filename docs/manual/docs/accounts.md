# Gestión de Cuentas

Verith te permite manejar múltiples cuentas financieras (Caja Chica, Banco Pichincha, Banco Guayaquil, etc.).

## Crear una Cuenta

1.  Ve a la pestaña **Cuentas**.
2.  Haz clic en **"Nueva Cuenta"**.
3.  Ingresa el nombre (ej: "Banco Principal") y el saldo inicial.

## Ver Saldos
