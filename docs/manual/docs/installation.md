# Instalación y Configuración

## 1. Instalación

1. Descarga el instalador `Verith_Setup.exe` desde nuestro sitio oficial.
2. Haz doble clic en el archivo y concede permisos de administrador si se solicitan.
3. Sigue el asistente de instalación.
   - _Nota:_ El instalador configurará automáticamente el motor de base de datos necesario. Esto puede tomar unos minutos la primera vez.

![Placeholder: Captura del Asistente de Instalación]

## 2. Primer Inicio

Por defecto se configurará un usuario administrador para que pueda ingresar.
Las credenciales de ingreso las podrá encontrar junto a la descarga del instalador

> Al abrir Verith por primera vez, se recomienda crear su propio usuario.

![Placeholder: Captura de pantalla de creación de primer usuario]

## 3. Configuración Inicial

Ve a la pestaña de **Configuración SRI** para ingresar los datos de tu empresa:

- RUC y Razón Social.
- Archivo de Firma Electrónica (.p12).
- Logo de la empresa.
