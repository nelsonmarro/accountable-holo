# Gestión de Clientes

Para emitir facturas electrónicas válidas, necesitas tener registrado a tu cliente con sus datos fiscales correctos. Verith te permite gestionar tu base de datos de clientes de forma sencilla.

## Acceder al Módulo

1.  Haz clic en la pestaña **"Clientes"** en la barra de navegación principal.
2.  Verás la lista de todos los contribuyentes registrados en tu sistema.

![Placeholder: Captura de la pantalla principal de Clientes]

## Crear un Nuevo Cliente

1.  Haz clic en el botón **"Nuevo Cliente"**.
2.  Llena el formulario con los datos requeridos:
    *   **Identificación:** RUC (13 dígitos), Cédula (10 dígitos) o Pasaporte.
    *   **Razón Social / Nombre:** Nombre legal del cliente.
    *   **Correo Electrónico:** **Crucial.** Aquí llegará automáticamente la factura electrónica (XML y PDF).
    *   **Dirección y Teléfono:** (Opcional) Datos de contacto adicionales.
3.  Haz clic en **"Guardar"**.

> **Nota:** El sistema valida automáticamente que la cédula o RUC sean algoritmicamente correctos para evitar rechazos del SRI.

![Placeholder: Captura del formulario de creación de cliente]

## Buscar y Editar

*   **Buscar:** Usa la barra de búsqueda superior para encontrar clientes por nombre o número de identificación.
*   **Editar:** Si un cliente cambia de correo o dirección, haz clic en el botón de **Editar** (lápiz) en su fila correspondiente.

## Consumidor Final

El sistema incluye por defecto al cliente **"CONSUMIDOR FINAL"** (Identificación: 9999999999999). Úsalo para ventas menores donde el cliente no requiera factura con datos.
