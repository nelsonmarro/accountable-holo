# Dashboard Financiero

El Dashboard es tu centro de mando. Aquí podrás ver el estado de tu negocio de un vistazo.

## Indicadores Clave (KPIs)

*   **Ingresos Totales:** Suma de todas las ventas del periodo seleccionado.
*   **Gastos Totales:** Total de egresos registrados.
*   **Utilidad Neta:** La diferencia real entre lo que entra y sale.
